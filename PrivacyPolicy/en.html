General Information

We can read your device hardware properties - based on that info we can

decide if your phone supports augmented reality or not.
Also we need to make sure that device has a camera.
 None of user hardware details, input information or anything else are going to

be saved anywhere.

Permissions

Camera - we use to scan qr codes, we not recording image from camera

while you use it and not send it anywhere either.


 Internet - we need it in order to download models from database, no 

personal information from the phone is sent to the web.

 Writing to local storage -  we need it to save atlass pages, no other files 

will be saved in your phone.
